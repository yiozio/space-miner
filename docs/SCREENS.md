# Space Miner — 画面仕様

各画面の概要・レイアウト・主要な操作を記述する。実装は `internal/scene` 配下で対応する Scene として表現される。

## ビジュアル方針
全画面共通のデザインルール。
- **暗い背景 + 単色線**で構成されたレトロベクター風 UI（古典的な宇宙シューターのトーン）
- 描画要素は基本的に「ライン」と「単色塗り」のみ。グラデーションや写実的なテクスチャは使わない
- 文字・アイコンも線で表現する
- 配色は「背景色」と「ライン色」の 2 色で構成され、テーマとして一括管理する（詳細は ARCHITECTURE.md「テーマシステム」参照）
- 設定画面で選択した背景色に応じてライン色も切り替わるため、ゲーム全体のトーンが連動する

## 画面一覧
| 画面 | 状態 | 概要 |
| --- | --- | --- |
| スタート | StateTitle | Continue / New Game / Load / Setting / Quit Game |
| 探索 | StateExploration | 宙域での操作・採掘・戦闘 |
| メニュー | StateMenu | ゲーム中のポーズメニュー（オーバーレイ） |
| 設定 | StateSettings | 背景色などのゲーム設定 |
| ワープ先選択 | StateWarpSelect | 移動先の宙域を選ぶ |
| 宇宙ステーション | StateStation | 売買・船改造・運送依頼 |
| 通信ダイアログ | StateDialog | 会話・イベント表示 |
| 全体マップ | StateWorldMap | 最後に入った FullMap の全域・ステーション・素材ゾーンを確認する |

## スタート画面
ゲーム起動直後に表示されるタイトル画面。中央に縦並びのメニューを表示する。

### 項目
| 項目 | 動作 |
| --- | --- |
| Continue | 最新のセーブデータでゲームを再開する。セーブが無い場合はグレーアウト |
| New Game | 新規にゲームを開始する |
| Load | セーブデータ選択画面を経由してゲームを再開する |
| Setting | 設定画面（StateSettings）に遷移する |
| Quit Game | ゲームを終了する |

### 操作
- 上下キー: 項目選択
- 決定キー: 項目実行
- Setting / Load から戻ると本画面に復帰する

## 探索画面
ゲームの主たる画面。プレイヤー宇宙船を中心に俯瞰視点で描画する。

### レイアウト
```
┌───────────────────┐
│ HUD（HP/シールド/燃料/資源）         │
│                                      │
│                                      │
│            （プレイ領域）            │
│             [Player]                 │
│                                      │
│                                      │
│                      ┌──────┐│
│                      │ ミニマップ ││
│                      └──────┘│
└───────────────────┘
```

### 要素
- **プレイヤー機**: 画面中央付近に固定。カメラがプレイヤーを追従
- **小惑星 / 惑星**: 採掘対象。HP を持ち、破壊で資源をドロップ
- **海賊船**: 敵エンティティ。撃破で破損モジュールをドロップ
- **HUD**: HP・シールド・燃料・主要資源量・装備モード
- **ミニマップ**: 画面右下。周辺宙域・ステーション・敵を簡易表示

#### 小惑星の構造

小惑星は複数のグリッドで構成され、グリッド毎にHPを持ち、破壊で資源をドロップする。 \
最大HPと資源の種類は、グリッドの色で判別する。

### 操作
- 移動: 推進・旋回
- 攻撃: 弾発射（自動照準パーツ装備時は自動）
- 高速ブースト: 燃料消費で短時間加速
- ワープ: 条件を満たせば WarpSelect へ遷移
- インタラクト: ステーションへのドッキング・運送依頼受領

## 全体マップ画面
探索画面から呼び出し、最後に入った FullMap を画面いっぱいに俯瞰表示する。スクロールせず、対象 FullMap 全域が常に画面に収まるよう自動でスケールする。表示中はゲーム内の時間を停止する（オーバーレイシーン）。

### 表示対象
- 「最後に入った FullMap」 1 つを対象とする
  - 区画外（FullMap の間の空虚）にいる場合も、直前にいた FullMap を表示し続ける
  - まだどの FullMap にも入っていない場合のみ「(NOT IN ANY FULL MAP)」を表示する
- 別の FullMap にワープ／到達したタイミングで対象が更新される

### レイアウト
```
┌──────────────────────────────────┐
│             WORLD MAP              │
│ ┌──────────────────────────────┐ │
│ │              ＋                │ │ ← FullMap 中心マーク
│ │     ◯ IRON     ◯ IRON+ICE     │ │ ← 素材ゾーン円（代表色）
│ │                                │ │
│ │       ◇ STATION                │ │ ← ステーション
│ │              ▲                  │ │ ← 自機現在位置（向きを示す三角形）
│ │     ◯ IRON                     │ │
│ │                ◯ BRONZE+ICE    │ │
│ └──────────────────────────────┘ │
│ [ M / Tab / Esc: Close ]           │
└──────────────────────────────────┘
```

### 要素
- **FullMap 外枠**: 矩形のライン枠。区画の境界を示す
- **中心十字**: FullMap の中心（典型的には基準ステーション）の位置マーク
- **素材ゾーン**: 円形ライン。代表色（重みが最大の素材色）で描画し、中央に素材ラベル（例: `IRON+ICE`）と素材色サンプルを並べて表示する
- **ステーション**: ひし形マーカー + `STATION` ラベル
- **自機現在位置**: 二等辺三角形マーカー（向きが分かるよう前方が頂点。FullMap 内にいるときのみ）

### 操作
- 探索画面で `M` または `Tab` を押すと開く
- `M` / `Tab` / `Esc` のいずれかで閉じ、探索画面に戻る

## ワープ先選択画面
別宙域への長距離移動を選ぶ画面。スターマップ風に行き先を表示する。

### 要素
- 既知宙域の一覧（未発見はマスクされた表示）
- 各宙域の燃料コスト・距離・主な資源・脅威レベル
- 確定／キャンセルで探索画面に戻る

## 宇宙ステーション画面
ハブ画面。以下のサブメニューを持つ。
- **資源売買**: 採掘した資源を換金
- **パーツ購入**: 通常パーツを入手
- **宇宙船改造**: グリッドにパーツを配置／取り外し
- **運送依頼ボード**: 別ステーションへの運送ミッション受注

## 通信ダイアログ
画面下部にウィンドウ表示する会話 UI。探索・ステーション双方からオーバーレイで呼び出される。
- 話者アイコン・名前・本文・選択肢
- イベントトリガー（クエスト発生、ストーリー進行）に使用

## メニュー画面
ゲーム中（探索・ステーション等）から呼び出されるポーズメニュー。直前の画面の上に黒い半透明背景を重ね、その上に中央寄せのメニューを表示する。表示中はゲーム内の時間を停止する。

### レイアウト
```
┌──────────────────────────────┐
│░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│ ← 直前画面 + 黒半透明オーバーレイ
│░░░░░░░░┌────────────┐░░░░░░░░│
│░░░░░░░░│   Save     │░░░░░░░░│
│░░░░░░░░│   Load     │░░░░░░░░│
│░░░░░░░░│   Setting  │░░░░░░░░│
│░░░░░░░░│ Quit Title │░░░░░░░░│
│░░░░░░░░└────────────┘░░░░░░░░│
└──────────────────────────────┘
```

### 項目
| 項目 | 動作 |
| --- | --- |
| Save | 現在のゲーム状態をセーブする |
| Load | セーブデータ選択画面を経由して再開する |
| Setting | 設定画面（StateSettings）に遷移する |
| Quit To Title | 確認モーダルを表示し、確定でスタート画面（StateTitle）へ戻る |

### 操作
- 上下キー: 項目選択
- 決定キー: 項目実行
- キャンセルキー（または同じキー再入力）: メニューを閉じて直前の画面に戻る
- Quit To Title 選択時は確認モーダル（はい / いいえ）をオーバーレイ表示

## 設定画面
ゲーム全体の設定を変更する画面。スタート画面・メニュー画面のいずれからも遷移可能。

### 項目
| 項目 | 内容 |
| --- | --- |
| テーマ（背景色） | 黒 / 紺 / 濃い緑 から選択。選択した背景色に応じてライン色も同時に切り替わる（テーマプリセット） |
| Back | 呼び出し元の画面（スタート画面 または メニュー画面）に戻る |

### 操作
- 上下キー: 項目選択
- 左右キー: 値の切り替え（テーマなど）
- 設定変更は即時反映し、永続化する
- Back（またはキャンセルキー）で呼び出し元の画面に復帰する

> 背景色とライン色はテーマプリセットとして対になっている。選べるのは「テーマ」単位であり、個別に色を組み合わせることはしない。
