下記の機能を実装したい。実装順は依存関係（土台 → 拡張）で並べてある。

## Phase 1: 世界の拡張
ワールドの構造拡張がほぼ全ての後続機能の土台になるので最優先。

[ ] 他のマップ、宇宙ステーションの追加
    - 既存の `entity.World` に FullMap を追加し、各マップに 1 つステーションを置く
[ ] 各マップで宇宙ステーション近くの背景に大きめの惑星または衛星を表示したい
    - 惑星、衛星は背景情報であり、ゲームプレイには影響しない
    - 惑星、衛星は背景の星の手前に表示する
    - 各 FullMap に紐づく天体データをここで定義（後続の宇宙マップとも共有）
[ ] 宇宙マップの実装
    - 恒星と惑星、衛星を表示
    - ワープ先選択 UI の母体になる
[ ] ワープ機能
    - 指定の宇宙ステーションに瞬時に移動する
    - 惑星、衛星には宇宙ステーションが１つある
[ ] ワープ時アニメーション
    - 自機が止まった状態で向きだけ移動先に合わせて、前から後ろに次々と線が移動してホワイトアウト
    - 急にホワイトアウトが消えて移動が完了する

## Phase 2: ダイアログ基盤
ストーリー・クエスト・NPC 会話の共通基盤。オープニングと宇宙酒場の前提。

[ ] 会話ダイアログ
    - 画面全体が薄暗くなり、画面下部にダイアログ、ダイアログ左にキャラクター画像があり、右にテキスト
    - 宇宙ステーション初回入船時に会話
[ ] 会話ダイアログ選択肢
[ ] オープニングストーリー
    - 画面いっぱいに画像を表示し、その上にキャラクター画像のないダイアログ

## Phase 3: 戦闘・クエスト
ダイアログが整ってから NPC・敵・依頼を載せる。海賊単体を先に作り、クエスト系で利用する。

[ ] 宇宙海賊の実装
    - 宇宙海賊出没エリアの実装
    - 出没エリアでは、敵の宇宙船が出没
    - 様々なパターンの宇宙船が自機に向かって銃を撃ってくる
    - 倒したらお金、ごく稀にパーツをドロップ
[ ] 宇宙酒場の実装
    - 自動生成のクエスト
      - 三つ表示
      - クエストクリアまたはお金を払って破棄したら次のクエストを表示
      - 要求されている資源を渡せばクエストクリア
      - クリア時、お金、またはパーツを取得
[ ] 宇宙海賊討伐クエスト
    - 指定マップの宇宙海賊を討伐するクエスト
    - 海賊（Phase 3 前段）と酒場クエスト基盤を組み合わせて実装

## Phase 4: 仕上げ・拡張
独立して進められる枝分かれ系。Phase 1-3 と並行可能だが、最後のポリッシュ枠に置く。

[ ] 武器の弾丸の種類を増やす
    - ボール型
    - レーザー型
    - 着弾時エフェクト大きさ（調整可能）
    - 線の太さを調整可能にして通常型にもバリエーションをつけたい
[ ] HUD調整
    - HP、シールドは自機の下にバーとして表示したい。バーはパーツが被らない位置に表示する。
