# Space Miner — ゲーム仕様

## 概要
Asteroid 風の俯瞰 2D 画面で宇宙空間を探索し、小惑星や惑星から資源を採掘して宇宙船を強化していくゲーム。

## 世界観
プレイヤーは個人採掘者として宇宙ステーションを拠点にし、近隣宙域から徐々に活動範囲を広げていく。海賊や運送依頼が存在する商業宙域が舞台。

## コアゲームループ
1. 宇宙ステーションから出航
2. 宙域を探索し、小惑星・惑星から資源を採掘する
3. ステーションに戻って資源を売却・パーツを購入・宇宙船を改造
4. より遠い宙域へワープし、活動範囲を広げる

## グリッドシステム
- 宇宙船のパーツ配置と小惑星の構造は共通のグリッドサイズを用いる
- 1 グリッド = 統一サイズの正方形セル。宇宙船パーツ・小惑星のブロックはいずれもこのセルを単位として構成する
- 共通サイズにすることで、当たり判定・描画・データ表現を一本化できる

## プレイヤー宇宙船
- グリッド状にパーツを組み合わせて構築する（グリッドサイズは「グリッドシステム」参照）
- パーツごとに性能（攻撃・推進・燃料・装甲・特殊機能 等）が異なる
- 海賊船から得られる「破損モジュール」が高性能パーツの入手経路となる

### パーツ凡例（サンプル図中で使用）
| 記号 | パーツ |
| --- | --- |
| `C` | コックピット（必須・1機に1つ） |
| `G` | 弾射出（前方固定砲） |
| `M` | 自動照準モジュール |
| `T` | 推進（スラスター） |
| `F` | 燃料タンク |
| `K` | カーゴ（資源収納） |
| `A` | 装甲 |
| `S` | シールド発生装置 |
| `W` | ワープドライブ |

### サンプル: スターター艇「Pebble」
最小構成。序盤に支給される標準採掘船。
```
┌─┬─┬─┐
│ │T│ │
├─┼─┼─┤
│G│C│G│
├─┼─┼─┤
│ │F│ │
└─┴─┴─┘
```
- 前方 2 門の手動照準で小惑星を砕く
- カーゴは無く、コックピットの内蔵スロットで運搬
- ワープ不可。近隣宙域専用

### サンプル: 採掘特化「Drill」
自動照準を載せた中盤の効率重視機。
```
┌─┬─┬─┬─┐
│ │T│T│ │
├─┼─┼─┼─┤
│G│C│M│G│
├─┼─┼─┼─┤
│G│K│K│G│
├─┼─┼─┼─┤
│ │F│F│ │
└─┴─┴─┴─┘
```
- 自動照準 `M` で近接小惑星に常時射撃
- カーゴ 2 マスで採掘量を確保
- 装甲なし。海賊宙域には不向き

### サンプル: 戦闘艇「Fang」
海賊狩り向け。装甲とシールドで打たれ強さを優先。
```
┌─┬─┬─┬─┐
│G│T│T│G│
├─┼─┼─┼─┤
│A│C│S│A│
├─┼─┼─┼─┤
│A│G│G│A│
├─┼─┼─┼─┤
│ │F│F│ │
└─┴─┴─┴─┘
```
- 4 門の砲と側面装甲で正面戦闘を想定
- シールドで瞬間火力を受け流す
- カーゴが少ないので破損モジュール回収用と割り切る

### サンプル: 運送艦「Hauler」
ステーション間輸送向け。カーゴとワープを最大化。
```
┌─┬─┬─┬─┬─┐
│ │T│T│T│ │
├─┼─┼─┼─┼─┤
│A│K│K│K│A│
├─┼─┼─┼─┼─┤
│A│K│C│K│A│
├─┼─┼─┼─┼─┤
│ │F│W│F│ │
└─┴─┴─┴─┴─┘
```
- カーゴ 5 マスで大量輸送
- ワープ `W` 搭載で遠距離ステーション間を直行
- 火力なし。護衛または回避前提

## 採掘システム
- 序盤: 宇宙船前面から弾を射出して小惑星を砕き、ドロップした資源を回収
- 中盤以降: 自動照準パーツを装備すると近接の小惑星を自動で攻撃
- 終盤: 完全自動採掘が可能になり、プレイヤーは戦略・運送・探索に集中できる

### 小惑星の構造
- 小惑星は複数のグリッドで構成され、グリッド単位で破壊・資源ドロップが発生する（グリッドサイズは宇宙船と共通、「グリッドシステム」参照）
- グリッドの色によって資源の種類と最大HPが決まる（色 = 資源種別の視覚表現）
- プレイヤーは見た目（色）から狙うべきグリッドを判断する

## 戦闘 / アクティビティ
- **小惑星採掘**: 基本の資源収集行動 
- **海賊船狩り**: 戦闘により破損モジュール（高性能パーツ素材）を入手
- **運送ミッション**: 宇宙ステーション間で資源を運搬し報酬を得る

## 移動システム
| 種類     | コスト         | 用途           |
|--------|-------------|--------------|
| 通常運行   | なし          | 宙域内での移動      |
| 高速ブースト | 燃料 + 推進パーツ  | 短時間での距離移動・回避 |
| ワープ    | 燃料 + ワープパーツ | 別宙域への長距離移動   |

## 進行（プログレッション）
- 初期: 宇宙ステーション近傍の小惑星採掘のみ
- 段階的に解禁: 高速ブースト → 海賊狩り → 運送 → ワープ → 自動採掘
- 解禁条件は資源・パーツ・ストーリー進行に紐づく
