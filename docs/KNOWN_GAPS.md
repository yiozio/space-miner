# 既知の未実装項目・将来課題

ADR は実装済みの設計判断のみを残す。ここでは未実装・暫定実装・将来検討項目を集約する。

## ゲーム進行・ループ
- **ゲームオーバー / 死亡処理**: `Player.HP` が 0 になっても何も起きない。リスポーン / Game Over 画面 / ペナルティを未実装。
- **燃料切れ時の挙動**: 通常加速は Fuel に依存しないため、燃料 0 でも基本航行は可能。ブーストだけが使えなくなる。
  「燃料切れ = 帰投不能」の緊張感は今のところ無い。
- **死亡 / 撃破に伴うアセットロス**: 海賊撃破は credits / パーツドロップで報酬になるが、
  プレイヤー側のロスト（積載資源の流出など）は無い。

## ステーション
- **修理 / 給油の対価**: `Repair` / `Refuel` ボタンは無料・即時。経済設計を入れたい場合は調整余地。
- **ドック離脱のアニメ**: 現在は瞬時に StationMenu を Pop するのみ。

## 戦闘
- **海賊 vs 海賊・海賊 vs 小惑星の物理**: 海賊はパススルー。同士討ちや小惑星衝突の緊張感を
  入れたい場合は AI と物理を拡張する。
- **海賊の死亡演出**: 単に消えるのみ。爆発エフェクトは未実装。
- **被弾無敵時間**: 撤去済み（ユーザー指示）。連続被弾で一気に削られる挙動を許容している。
- **シールド表示の詳細**: 現状は外周ラインで「張られている」ことだけ表現。残量変化のフィードバックは
  バーに依存する。

## クエスト・経済
- **複合報酬**: Quest の報酬は credits + 最大 1 種パーツ。複数同時付与や大量セットの仕組みは無い。
- **ステーション間運送ミッション**: TODO 案にあったが未実装。Tavern の Delivery は同ステーション内納品のみ。
- **クエスト失効**: 期限はない。受けたまま放置可能。
- **掲示板の枯渇**: スロットは CLEAR / DISCARD で必ず差し替わるため、掲示板が空になることはない。

## ダイアログ・ストーリー
- **本格的なストーリー進行**: 入船時の初回挨拶のみ。フラグ管理や条件分岐の状態機械は未整備。
- **多言語対応**: スクリプトはハードコード（英語）。
- **キャラクター画像**: ベクター固定 4 スタイル。外部画像化やアニメーションは未対応。

## ワープ
- **燃料コスト**: 仕様上は燃料 + Warp パーツが必要だが、現状は燃料消費なし（Warp パーツ装備のみ要件）。
- **ワープ先発見 / 解禁**: すべての FullMap が最初から見える。発見・到達によるアンロックは未実装。
- **複数恒星系**: World は 1 つの恒星系を前提。別恒星系（遠方への大規模ワープ）は ADR 0003 上拡張可能だが UI 未対応。

## UI
- **設定の項目数**: 現状はテーマ切り替えのみ。音量・キーバインド・難易度等は未実装。
- **キーバインド変更**: 不可。`internal/scene` 内でハードコード。
- **コントローラ対応**: 未対応（キーボード前提）。
- **ヘルプ画面**: 操作ヒントは画面下のテキストのみ。集約画面は無い。

## 技術
- **テスト**: ユニットテストは未整備。
- **アセットローダ**: 外部画像 / 音声を読み込む仕組みはまだ無い。すべて手続き的描画。
- **音 / BGM**: 未実装。
- **`PartID` の互換性**: enum を末尾追加でしか拡張しない規約。中央で再利用すると旧セーブが壊れる。

## 描画
- **HUD でのプレイ時間表示**: セーブメタには Playtime があるが、ゲーム中の HUD には表示していない。
- **ダメージ点滅**: 自機・海賊への被弾フィードバックは色フラッシュ等が無く、HP バーの減りで判断する。
- **最低限のスクリーンシェイク等**: 演出として未導入。

---
ここに記された項目は意図的な未実装であり、ADR の決定事項とは別に追補として管理する。
新たに着手する場合は対応する ADR を新設するか、既存 ADR に追記する。
