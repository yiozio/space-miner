# プロジェクト概要
2D宇宙採掘ゲーム「Space Miner」の開発。
ゲーム仕様・アーキテクチャ・画面構成の詳細は `docs/` 配下を参照すること。

# ビルド確認
ビルド確認には必ず `go vet ./...` を使用すること

# フォーマット
実装が終わったら必ず `gofmt -w .` を実行すること

# 技術スタック
- 言語: Go 1.22+
- ゲームエンジン: ebitengine v2

# コーディング規約
- Effective Goに倣うこと
- コメントは日本語で行うこと
- パッケージ構成はクリーンアーキテクチャを意識する
- ゲームの状態管理はステートマシンパターンを使用する

# 参照すべきドキュメント
- docs/GAME_DESIGN.md: ゲーム全体の仕様・世界観・ストーリー
- docs/ARCHITECTURE.md: システムアーキテクチャ・状態遷移・ディレクトリ構成
- docs/SCREENS.md: 各画面の詳細仕様
